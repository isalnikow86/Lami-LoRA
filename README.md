# Lami-LoRA — LeoLM 7B LoRA Training (Klexikon Dataset)

## 🔍 Overview

This repo trains a **LoRA adapter** on the **LeoLM/leo-hessianai-7b** model using **Klexikon data** (German children's encyclopedia) on **H100** GPU.
